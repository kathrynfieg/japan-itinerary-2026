# Japan 2026 Itinerary

Personal trip page for Zac, Jess & Kat — Tokyo, Hakone, Kyoto & Osaka (21–31 July 2026).

Hardcoded itinerary built with Vue 3, TypeScript, Vite, and Tailwind CSS.

## Setup

Requires [Node.js](https://nodejs.org/) (v20+ recommended) and [pnpm](https://pnpm.io/).

```bash
pnpm install
```

## Run

Start the local dev server:

```bash
pnpm dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

## Editing the trip

Day-by-day plans, flights, and activities live in:

```
src/data/trip.ts
```
